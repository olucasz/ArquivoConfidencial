# Arquivo Confidencial

Experiência digital interativa criada em React + Vite.

## Rodar localmente

```bash
npm install
npm run dev
```

## Gerar build

```bash
npm run build
```

## Deploy GitHub Pages

```bash
npm run deploy
```
